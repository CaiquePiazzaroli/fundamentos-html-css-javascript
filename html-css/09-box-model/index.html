Box model padrão e box model border box

Com relação ao tamanho do box model, podemos definir dois tipos: o padrão (que não inclui o padding e border em seu tamanho) e o box model (em que seu tamanho ja inclui o padding e a margin), vamos aos exemplos

Isso é um paragrafo que possui 440px de tamanho + 5px de padding e mais 7px de margin. Seu tamanho total é de 440+5+7 = 452px (é preciso somar os valores)

Por outro lado, este parágrafo foi definido com o tamanho 440px, e por conter a propriedade box-sizing:border box, passa a incluir dentro desses 440px os tamanhos do padding e da border. Resumindo, seu tamanho real é 440, não sendo necessário somar qualquer valor da borda ou padding.

Tentando aplicar padding, margin, width e hight ao span dentro de um parágrafo, lembrando que a tag span tem o display do tipo inline!

Tentando aplicar padding, margin, width e hight ao span dentro de um parágrafo, alterando o diplay de inline para inline block, que é um meio termo entre inline e block (padding, margin, width e high são respeidados)
