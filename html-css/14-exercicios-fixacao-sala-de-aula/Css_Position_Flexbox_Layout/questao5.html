Resposta

A Posição static é o padrao do html e não é influenciado pelos e atributos Top, botton, left e right etc. Sendo assim, esta div se encontra na posição "normal".
A Posição relative é a posição do html que usa como ponto inicial o estado "normal" do elemento, ou seja, o posicionamento utilizando os valores top, bottom, left e right serão aplicados de acordo com seu estado inicial. Além disso, o espaço inicial de onde esse elemento saiu nao pode ser ocupado por outro elemento block, como no caso da div absolute.
O posicionamento absolute considera o elemento pai como refencia para se posicionar, desde que esse elemento pai não seja de posicionamento static. Nesse caso, o espaço inicial do elemento, diferentemente de relative, é desconsiderado.
O posicionamento fixed toma como referencia a viewport do usuário, ou seja, fica na tela o tempo todo. E bem parecido com o absolute mas nesse caso o fixed toma como refecia a viewport.
