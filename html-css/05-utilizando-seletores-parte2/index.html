Seletores de atributos

Este paragrafo está sendo estilizado utilizando o seletor de atributo title, todos os paragrafos com o atributo title serão estilizados. Mesmo que o valor dos atributos sejam diferentes.

Neste caso, o background color vem da regra que seleciona esse objeto pelo atributo: title, e a borda é estilizada utilizando o atributo title="atributos2". Que considera o valor do atributo.

Esse elemento foi estilizado utilizando o seletor de atributos p[class~="f"] {exemplo}, que verifica se na classe contém a letra f e, se caso existir, aplica o estilo.

Seletores de atributos
Seletor Exemplo Descrição
[attr^=Value] li[class^="box-"] Seleciona o elemento que possui um atributo classe e que a classe comece com o valor especificado Nesse caso: seleciona o elemento que possui a classe que tem em seu valor o início "box-"
[attr$=value] li[class$="-box"] Seleciona o elemento que possui um atributo classe e que a classe termine com o valor especificado Nesse caso: seleciona o elemento que possui a classe que tem em seu valor no final "-box"
[attr*=value] li[class*="box"] Seleciona o elemento que possui um atributo classe e que a classe contem em qualquer lugar da string o valor especificado Nesse caso: seleciona o elemento que possui uma classe que contem a palavra "box"
Observação Os seletores acima são case sensitive, para desativar o case sensitive deve-se adicionar o i na frente do valor que se deseja selecionar. Exemplo: li[class^="a" i] (seleciona os li que possuem a letra "a" ou "A")

Seletores pseudo-classes

São aqueles nos quais selecionam os obejtos de acordo com seu estado, essa tag p, muda de cor ao passar com o mouse por cima.

Existem ainda seletores mais específicos que selecionam partes especificas de um elemento. São chamados de pseudo elementos pois agem como se existisse uma tag html, mas não existe, como a primeira linha desse texto por exemplo. Neste caso, o seletor ::first-line, seleciona apenas a primeira linha do elemento do paragrafo!

Seletores pseudo-classes são importantes para a manutenção do código! Existem muitos outros tipos que devem ser consultados, mas listaremos abaixo alguns exemplos:

Lista basica de pseudo-classes
Pseudo-Classe Descricão
:first-child Seleciona o primeiro elemento que aparece em uma tag html
:hover Seleciona o elemento quando o mouse está posicionado sobre ele
:focus Quando o usuário está interagindo efetivamente com o elemento html. Ex: click do usuario
::before É um pseudo elemento que permite adicionar conteúdo ao html utilizando css, nesse caso, é adicionado antes do elemento html
::after É um pseudo elemento que permite adicionar conteúdo ao html utilizando css, nesse caso, é adicionado depois do elemento html

Isto é um exemplo de utilização de after e before

Também é possivel juntar pseudo-classes com pseudo elementos:

Exemplo de utilização de pseudo-classes com pseudo-elementos
Seletor Descrição
article p:first-child::first-line Seleciona a primeira linha do primeiro elemento p que está na tag html article

Combinadores

São aquele seletores que combinam outros seletores para selecionar um elemento específico. Para estilizar este paragrafo foi utilizado ".combinadores > p" que seleciona todos os elementos p dentro da div com a classe combinadores

Combinador Exemplo Descrição
Combinador descendente body article p Seleciona todos os elemenos que são descendentes de uma tag, mesmo que não seja descendentes diretos.
Combinador filho article > p Seleciona apenas os descendentes diretos da tag, se houver outro paragrafo dentro de uma div nessa mesma tag article, não será selecionada.
Combinador próximo irmão h1 + p Seleciona a próxima tag que está no mesmo nível. No exemplo dado: o seletor aplicaria o css para a próxima tag de paragrafo que aparece após a tag h1.
Combinador subsequentes irmãos h1 ~ p Seleciona todas as proximas tags p's. No exemplo dado: o seletor aplicaria o css para todas tags de paragrafos que aparece após a tag h1.

Combinadores podem ser aninhados para uma melhor leitura

Maneiras diferentes de escrever a mesma coisa
Aninhado Não aninhado 
p {
    ~ img {
    }
}
p ~ img { } 
p {
    & img {
    }
}
p img { }
